ReadMe

Arrow Keys move you.

When you load, you need to back up to see the initial scene.

You load in looking in the POSITIVE Z direction, so your X directions will be backwards.

Object EasterEggs: Name an object "Ground" and it loads a grass texture. Name an object "Sky" and it loads a stars texture.

I recommend Position(0,-1,0) Scale(100,.1,100) for the ground.

I recommend Position(0,0,0) Scale(200,200,200) for the sky.

When creating an object it makes it at the origin it only takes into account the name field and the object type (cube or sphere).

Loaded with a semi-scale solar system. It's close, but not perfect.
